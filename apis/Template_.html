templateWithComponent(requireFunction, moduleId, callback)

Creates a (or uses a previously created one) Template object out of a fully instantiated component. This means creating a markup from the Component object's element and a serialization with the Component object as the owner of the Template. This function guarantees that only one Template object is used per type of component reducing the number of objects needed.

Parameters:
Name Type Description
requireFunction Function

The require function to use to load the modules used in the serialization.
moduleId String

The module id.
callback Function

The function to call when the template is ready, receives a Template object as a parameter.
