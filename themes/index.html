Montage Themes

When building a Web App with MontageJS, you can use the following UI component sets.

Digit

Digit is a is touch-optimized components set. Use it to create Web Apps for tablets and phones.

Digit also comes with 3 skins: A basic flat one, light and dark skin.

See on GitHub

Note: Development is still in progress, below a preview with some upcoming components.

Matte

It's a basic, neutral-looking theme for HTML5 controls and inputs. You can use it as a wireframe theme for prototypes, starting point for creating your own theme, or easily style it with CSS to create custom skins.

See on GitHub

Native

Native components use the browser's default styles. This set is great if you wanna custom design everything using your own CSS.

See on GitHub
