To begin, download and install Node.js if you haven't already.

Although Montage does not rely on Node.js, it does depend on the Node Package Manager (NPM) for fulfilling dependencies throughout development. Besides, tools and utilities are written in JavaScript and executed using Node.js so it's useful to have installed.

Next, follow these steps to set up your Montage development environment:

  1. Install minit, the Montage Initializer:

    $ sudo npm install -g minit@latest

  2. Create your first app:

    $ minit create:app --name hello
$ cd hello

  3. Serve up your new application for development (or use your own web server):

    $ python -m SimpleHTTPServer

    and point your browser to http://localhost:8000/.

The server is required because Montage uses XHR to load files during development, which cannot be done on the file:// protocol.
